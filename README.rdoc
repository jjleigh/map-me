Welcome to Map-me

This is a rails based application which functions to map a users social media followers.

Heroku host: http://map-me2.herokuapp.com

Functionality Highlights

Maping functionality (powered by geocoder gem and google maps API)

3rd party user authentication (powered by gems: omniauth, omniauth-twitter, omniauth-facebook, omniauth-instagram, omniauth-google-oauth2)

Responsive framework (ZURB Foundation)
